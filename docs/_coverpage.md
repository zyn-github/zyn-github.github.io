# 张十一

> 尸位素餐，沐猴而冠

- 『人为善，福虽未至，祸已远离；人为恶，祸虽未至，福已远离；行善之人，如春园之草，不见其长，日有所增；作恶之人，如磨刀之石，不见其损，日有所亏。』
- 《春秋·曾子》
